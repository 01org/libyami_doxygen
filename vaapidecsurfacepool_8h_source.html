40 /***
41  * \class VaapiDecSurfacePool
42  * \brief surface pool used for decoding rendering
43  * <pre>
44  * 1. the surface status is described by 2 bitwise flag: | SURFACE_RENDERING | SURFACE_DECODING |
45  * SURFACE_DECODING is set when the buffer is used for decoding, usually set when decoder create a new #VaapiPicture.
46  * SURFACE_DECODING is cleared when decoder doesn't use the buffer any more, usually when decoder delete the corresponding #VaapiPicture.
47  * SURFACE_TO_RENDER is set when #VaapiPicture is ready to output (VaapiPicture::output() is called).
48  * SURFACE_TO_RENDER is cleared when VASurface is sent to client for rendering (VaapiDecoderBase::getOutput())
49  * SURFACE_RENDERING is set when VASurface is sent to client for rendering (VaapiDecoderBase::getOutput())
50  * SURFACE_RENDERING is cleared when the surface is returned back from client (VaapiDecoderBase::renderDone())
51  * if no flag is set, the buffer/surface can be reused -- associate with a new VaapiPicture
52  * 2. the free surface is in a first-in-first-out queue to be friendly to graphics fence
53  * 3. most functions in this class do not support multithread except recycle.
54  * 4. flush need called in decoder thread and it will make all following acuireWithWait return null surface.
55  * until all surface recycled.
56  *</pre>
57 */
