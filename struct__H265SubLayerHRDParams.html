Detailed Description

H265SubLayerHRDParams: :togeter with bit_rate_scale, it specifies the maximum input bitrate when the CPB operates at the access unit level : is used together with cpb_size_scale to specify the CPB size when the CPB operates at the access unit level : is used together with cpb_size_du_scale to specify the CPB size when the CPB operates at sub-picture level : together with bit_rate_scale, it specifies the maximum input bit rate when the CPB operates at the sub-picture level : indicate whether HSS operates in intermittent bit rate mode or constant bit rate mode.
