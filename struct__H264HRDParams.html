Detailed Description

H264HRDParams: : plus 1 specifies the number of alternative CPB specifications in the bitstream : specifies the maximum input bit rate of the SchedSelIdx-th CPB : specifies the CPB size of the SchedSelIdx-th CPB bit_rate_value_minus1: specifies the maximum input bit rate for the SchedSelIdx-th CPB : is used together with cpb_size_scale to specify the SchedSelIdx-th CPB size : Specifies if running in itermediate bitrate mode or constant : specifies the length in bits of the cpb_removal_delay syntax element : specifies the length in bits of the dpb_output_delay syntax element : >0 specifies the length in bits of the time_offset syntax element. =0 specifies that the time_offset syntax element is not present : Length of the time offset
