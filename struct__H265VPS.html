Detailed Description

H265VPS: : vps id : should be zero, but can be other values in future :specifies the maximum number of temporal sub-layers : specifies whether inter prediction is additionally restricted : ProfileTierLevel info : indicates the presense of vps_max_dec_pic_buffering_minus1, vps_max_num_reorder_pics and vps_max_latency_increase_plus1 : specifies the maximum required size of the decoded picture buffer : indicates the maximum allowed number of pictures that can precede any picture in the CVS in decoding order : is used to compute the value of VpsMaxLatencyPictures : specifies the maximum allowed value of nuh_layer_id : specifies the number of layer sets : specifies whether a nuh_layer_id included in the layer identifier list : indicate the presence of num_units_in_tick, time_scale, poc_proportional_to_timing_flag and num_hrd_parameters : number of time units in a tick : number of time units that pass in one second : indicate whether the picture order count is proportional to output timin : specifies the number of clock ticks corresponding to a difference of picture order count values equal to 1 : number of hrd_parameters present : index to the list of layer hrd params. : the H265HRDParams list
